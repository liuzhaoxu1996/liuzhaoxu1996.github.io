# 37. 引用的缺省值 null

# 定义

  • null 代表空，不存在，可以读作空
  • 引用类型的数组创建出来，初始值都是空

# null 带来的问题

  • NullPointerException(NPE)

举个例子：

double myList;
myList.name = 'a';
